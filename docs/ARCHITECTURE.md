# Enterprise Agent Platform 架构文档

## 🏗️ 整体架构

Enterprise Agent Platform 采用分层架构设计，基于AutoGen for .NET框架，专门针对国产模型和Ollama进行优化。

```
┌─────────────────────────────────────────────────────────────┐

                                │
